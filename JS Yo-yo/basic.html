Yomega Yo Yo With a Brain:


The Yomega Brain contains a centrifugal clutch that opens on the down swing, enabling the yo-yo to "sleep" or spin at the end of the string. When the yoyo slows down, the clutch snaps shut causing the yoyo to return automatically to the player's hand. The Yomega Brain is one of the most popular yoyo designs in history and great for beginners.

Spesification:

YoYo Diameter : 58.79mm
YoYo Width : 35.72mm
YoYo Weight : 59.3 grams
YoYo Material : Plastic
YoYo Bearing : Yomega Plastic Transaxle
YoYo Response : Starburst

Yomega Power Brain XP:


The Yomega Power Brain is two high performance yoyos in one! Designed for both beginner and intermediate play, this yoyo uses a "Smart Switch" to enable the player to chose between automatic and manual function with the flick of a switch. Beginners can easily get a sense of accomplishment while they learn to handle the yoyo in the automatic return mode. As their skills increase, they can switch to manual mode for longer, faster spin times and tricks that are more challenging.

Spesification

Weight (g) : 68.4
Diameter (mm) : 60.20
Width (mm) : 35.9
Body Material : Plastic (Polycarbonate/ABS/PS)
Bearing Type : Plastic Spool
Response : Starburst

Yomega Alpha Wing:

Introducing Yomega's all new, value priced string trick yoyo! The Alpha Wing features a flared body shape, wide string gap and steel axle - perfect for new players to learn basic string tricks! Now you can enjoy legendary Yomega quality and performance at a great low price.
